## 💻 API快速参考

### 接口1：POST /api/v1/pest/identify
**用途**：识别单张图片的病害  
**输入**：multipart/form-data (image + region)  
**输出**：JSON (disease_type, confidence, suggestion)  
**响应时间**：200-300ms  
**成功率**：95%+  

### 接口2：POST /predict_batch
**用途**：批量处理多张图片  
**输入**：JSON (image数组，base64编码)  
**输出**：JSON (批量结果数组)  
**性能**：批处理提速40%  

### 接口3：GET /health
**用途**：健康检查  
**返回**：{"status": "healthy", ...}  
**用于**：容器编排、负载均衡器探针  

### 接口4：GET /model_info
**用途**：获取模型元数据  
**返回**：模型类型、支持的病害列表

## 🛠️ 故障排除

### 问题1：模型加载失败
```
❌ 错误: ModuleNotFoundError: No module named 'ultralytics'
✅ 解决: pip install ultralytics==8.0.196
```

### 问题2：GPU内存不足
```
❌ 错误: CUDA out of memory
✅ 解决: 
   - 使用CPU: python start.py --cpu
   - 减少batch size
   - 启用模型量化
```

### 问题3：模型文件找不到
```
❌ 错误: FileNotFoundError: best.pt not found
✅ 解决:
   - 检查: runs/detect/train/weights/best.pt
   - 下载: 从项目云存储下载
   - 训练: python train.py
```

### 问题4：请求超时
```
❌ 错误: Request timeout
✅ 解决:
   - 增加超时时间: timeout=300
   - 减少图片大小
   - 添加Nginx缓存
```
