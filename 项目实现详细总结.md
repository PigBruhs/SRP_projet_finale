## 六、使用示例

### Python客户端调用
```python
import requests

# 上传图片识别
with open("rice_image.jpg", "rb") as f:
    files = {"image": f}
    data = {"region": "浙江杭州"}
    response = requests.post(
        "http://localhost:5000/api/v1/pest/identify",
        files=files,
        data=data
    )

result = response.json()
if result['code'] == 0:
    print(f"病害类型: {result['data']['disease_type']}")
    print(f"置信度: {result['data']['confidence']}")
    print(f"严重程度: {result['data']['severity']}")
    print(f"防治建议: {result['data']['suggestion']}")
```

### JavaScript客户端调用
```javascript
async function identifyDisease(imageFile) {
    const formData = new FormData();
    formData.append('image', imageFile);
    formData.append('region', '浙江杭州');
    
    const response = await fetch(
        'http://localhost:5000/api/v1/pest/identify',
        { method: 'POST', body: formData }
    );
    
    return await response.json();
}

// 使用
const result = await identifyDisease(imageFile);
console.log(`识别结果: ${result.data.disease_type}`);
```

## 七、性能瓶颈与优化空间

### 当前瓶颈
1. **GPU显存限制** - 单张图片处理时显存占用1.2GB
   - 优化：使用TensorRT量化模型
   
2. **漏检率** - Recall 56.4%，意味着约44%的病害未被检测
   - 优化：增加训练样本、调整置信度阈值
   
3. **极端条件** - 逆光、遮挡等复杂环境识别准确度降低
   - 优化：添加数据增强、多光照训练样本

### 优化建议
| 优化项 | 预期提升 | 实施难度 | 时间投入 |
